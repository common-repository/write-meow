Cement your love for injecting meow inside of sentences, by replacing all instances of "now" with "meow" in your site content!

== Description ==

This plugin will show your love for slipping the word meow inside sentences without people noticing.

Install and activate it.  Right meow.

Fully compatible with:

* [The Events Calendar Shortcode PRO][tecs]
* [Event Calendar Newsletter][ecn]

[tecs]: https://eventcalendarnewsletter.com/the-events-calendar-shortcode?utm_source=wordpress.org&utm_medium=link&utm_campaign=wm-readme&utm_content=description
[ecn]: https://eventcalendarnewsletter.com/?utm_source=wordpress.org&utm_medium=link&utm_campaign=wm-readme&utm_content=description

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/write-meow` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Enjoy!
